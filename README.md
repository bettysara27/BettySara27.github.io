## 🚀 Deploy on GitHub Pages (step-by-step)

1. Create a new GitHub repository named `bettilda-portfolio` (or any name you prefer)
2. Upload all files in this folder to the repository root
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose branch: `main`, folder: `/ (root)`
6. Click **Save**
7. Your site will be live at: `https://<your-github-username>.github.io/bettilda-portfolio/`
